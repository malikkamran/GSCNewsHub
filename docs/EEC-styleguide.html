EEC Design & Typography Styleguide

1) Font Families and CSS Properties

  • Inter: global sans for body/headings; sizes 16/18/20/24/30/38/46; weights 400/500/600
  • Roboto: form selects/controls; size 14; weight 300; color #142642
  • Body: Inter 16px 400; line-height ~1.5
  • Headings: H1/H2 at 600; H3/H4 at 500

2) Visual Hierarchy & Typography Scale

Base 16px with scale: 18/20/24/30/38/46. Weights 600/500/400 aligned to hierarchy. Small text at 12–14px.

3) Patterns: Headers, Body, Interactions

  • Headers: strong size/weight contrast, consistent spacing
  • Body: Inter 16px 400, comfortable line-height
  • Buttons/Inputs: token-driven colors, transitions ~0.2–0.3s, visible focus
  • Tabs/Cards: hover emphasis, subtle elevation

4) Color Palette (Hex) & Usage

#162741 Primary
#B58E46 Gold
#FCFCFC White
#737D8D Gray
#EDF0F9 BG Blue
#F6F6F6 BG Gray
#DDDDDD Line

5) Spacing & Layout Grid

Spacing: 4/8/12/16/20/24/32/48px. Grid: 2–5 columns responsive; breakpoints ~430/767/1023/1279/1280/1480; containers mix % and fixed widths.

6) Emotional Impact & Brand

Formal, trustworthy, authoritative. Blue-centric palette with gold accents; modern clarity via Inter; maintain AA contrast.

Implementation

@import url("https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600&display=swap");
.eec-theme body{font-family:Inter,sans-serif}
h1{font-weight:600;font-size:46px}
h2{font-weight:600;font-size:38px}
h3{font-weight:500;font-size:30px}
body{font-weight:400;font-size:16px}
.small{font-size:14px}
.btn-primary{background:#162741;color:#FCFCFC;padding:12px 20px}
